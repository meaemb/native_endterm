# Campus Companion

## Short Description
Browse TV shows using the TVMaze API, save favorites, and discuss shows in real time.

## Full Description
Campus Companion is a mobile application built with Kotlin and Jetpack Compose.
The app allows users to search for TV shows using the TVMaze public API, browse
show details, save favorites, and participate in real-time discussions.

Key features include offline-first caching, pagination, debounced search,
Firebase authentication, and real-time comments.

## Screenshots
- Feed screen with TV shows
- Details screen with image and description
- Search screen with debounced input
- Offline mode with cached content

## Privacy
- Authentication via Firebase Authentication
- User-specific data stored securely in Firebase Realtime Database
- No sensitive personal data shared with third parties

## App Identity
- App name: Campus Companion
- Package: com.example.campuscompanion

## Data Collected
- Email (Firebase Auth)
- Favorites (per user)
- Comments (per user)
- No payment data
- No location data
